visual_behavior_analysis
========================

Analysis package for the `Allen Brain Observatory: Visual Behavior 2P <https://portal.brain-map.org/explore/circuits/visual-behavior-2p>`_ project.

This package provides tools for loading, processing, and analyzing behavioral and two-photon calcium imaging data from the Visual Behavior dataset. It wraps the `AllenSDK <https://github.com/AllenInstitute/AllenSDK>`_ and extends it with purpose-built analysis utilities for response quantification, dimensionality reduction, quality control, and visualization.
